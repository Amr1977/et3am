# Placeholder Sound Files

Place your MP3 sound files here. Files should be named:

- new_meal.mp3 - Played when a new donation is available
- reserved.mp3 - Played when your donation is reserved
- delivered.mp3 - Played when a donation is completed
- message.mp3 - Played when a new chat message arrives
- cancelled.mp3 - Played when a reservation is cancelled

If files are missing, the app will use fallback tones automatically.

Just replace these placeholder files with your own MP3 files.
