# Project Analysis: et3am - Food Donation Platform

## Project Overview

**et3am** is a food donation platform with full i18n support (English & Arabic with RTL). The project is a full-stack web application with:

- **Frontend**: React + Vite + TypeScript + react-i18next
- **Backend**: Express + TypeScript + PostgreSQL + Firebase
- **Auth**: Firebase Auth (frontend) + JWT (backend)

## Architecture


## Tech Stack

- Express 4.18.2
- PostgreSQL (pg 8.20.0)
- Firebase Admin 12.0.0
- Passport.js with Google OAuth
- JWT, bcryptjs

## Key Features

1. **Food Donations**: Create, browse, and claim food donations
2. **Location-based**: Maps with clustering, geocoding
3. **i18n**: English/Arabic with RTL support
4. **Authentication**: Firebase Auth + JWT tokens
5. **Privacy**: Donor/receiver details protected, Instapay support
